## Instalaciones desde VSCode

- Instalar Python

```bash
$ pip install python
```
- Instalar Django
```bash
$ pip install django
```
- Instalar Crispy-forms
```bash
$ pip install django_crispy-forms
```
- Instalar Pilow
```bash
$ pip install pillow 
```
- Instalar ckeditor
```bash
$ pip install django-ckeditor
```

## Verificar que este todo correctamente instalado

- Python en el bash
```bash
$ python --version
Python 3.10.7
```
- Django en el shell
```ps
>>> import django
>>> django.VERSION
(4, 1, 2, 'final', 0)
>>>
```
